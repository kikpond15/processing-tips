# Processingでよく使うショートカットや設定
---
## よく使うショートカット
- 実行（Mac：command+'r'　/ Win：control+'r'）
- 停止（escape）
- 保存（Mac：command+'s'　/ Win：control+'s'）
- オートインデント（Mac：command+'t'　/ Win：control+'t'）
- コメントアウト（Mac：command+'/'　/ Win：control+'/'）

---
